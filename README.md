
# PROYECTO FINAL – MÉTODOS NUMÉRICOS

Este proyecto presenta una aplicación web interactiva que implementa tres temas principales del Análisis Numérico, cada uno aplicado a problemas reales:

1. PROPAGACIÓN DEL ERROR (AGRICULTURA)
Se analiza cómo el error en las mediciones de masa y volumen afecta el cálculo de la concentración de nitrógeno en el suelo. El sistema calcula automáticamente la incertidumbre total mediante propagación de errores y muestra el impacto de la imprecisión de instrumentos en decisiones agrarias.

2. BÚSQUEDA DE RAÍCES (ROBÓTICA)
Se simula el cálculo del ángulo necesario para que un robot lanzador alcance un punto específico (x,y). Se usa el método de la bisección para resolver la ecuación no lineal que modela la trayectoria del proyectil. Esto permite encontrar la solución numérica al problema de trayectoria.

3. INTEGRACIÓN NUMÉRICA (ENERGÍA SOLAR)
Se aplica el método del trapecio para estimar la energía total recibida por un panel solar durante un día completo. La irradiancia solar se modela como una función continua y se integra para obtener la energía diaria.

El sistema fue diseñado con campos de entrada editables que permiten experimentar con valores reales y observar directamente los resultados.

Este trabajo demuestra la utilidad de los métodos numéricos en problemas de ingeniería, agricultura, robótica y medio ambiente.
